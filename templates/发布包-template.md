## ⑤ 发布前自检清单

**视觉**
- [ ] 9 张视觉尺寸严格一致
- [ ] 封面 / 末页大字 ≥ 156 px
- [ ] 中间页正文 38-44 px(不能全大字)

**字数硬约束**(v2 新增 · 见视觉规范.md 第 11 节)
- [ ] 总字数 ≤ 1000(中文)
- [ ] 封面主标 ≤ 15 字 + 副标 ≤ 25 字
- [ ] 中间页(02-06, 08)每张 ≤ 180 字
- [ ] 金句页(07)≤ 30 字 / CTA 页(09)≤ 80 字

**饶秋文字风格**(画像第 5 节红线)
- [ ] 没出现"赋能 / 抓手 / 闭环 / 颗粒度 / 链路"
- [ ] 没用"是 X 不是 Y"句式
- [ ] 没用"今天就能带走的一句话"式 CTA
- [ ] **没有 emoji**(只允许米金 ✓ 和灰色 ✗)

**🔴 小红书合规红线**(踩任一限流 / 封号)
- [ ] 全部图 + 配文都不出现"公众号 / 视频号 / 抖音"等其他平台名
- [ ] 没有"私你 / 私我 / 加我"等诱导私信
- [ ] 没有具体微信号 / 公众号号 / 邮箱
- [ ] 标签不含 #公众号 #微信 等导流词

**闭环 / 互动**
- [ ] 标题 ≠ 封面文案(独立写,不复制封面)
- [ ] 正文有开放式问题
- [ ] CTA(第 9 张)只用 关注 / 收藏 / 评论 三种之一

**饶秋两个必问**(画像第 4.4 节)
- [ ] 这是真实的饶秋吗?(拍着别扭就别发)
