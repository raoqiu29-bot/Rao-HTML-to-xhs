# 小红书合规红线

> 踩任一条 = 限流 / 封号。每篇发布前过一遍。

## 🔴 绝对禁区(不能出现在图里 / 文案里 / 标签里)

### 1. 站外引流类

| 词 | 为什么禁 |
|---|---|
| "公众号 / 视频号 / 抖音 / B 站" | 平台名 = 直接判定站外引流 |
| "全平台同名 / 全网账号 X" | 暗示站外账号 |
| "#公众号 #微信" | 标签级引流 |

### 2. 诱导私信类

| 词 | 为什么禁 |
|---|---|
| "评论区扣 XX,我私你 / 我发你 / 我私聊" | 诱导私信 |
| "加我微信 / 加我 V / 主页加好友" | 诱导加微信 |

### 3. 联系方式类

| 词 | 为什么禁 |
|---|---|
| 具体微信号 / 公众号号 | 引流 |
| QQ / 邮箱 / 群号 | 引流 |
| "看主页有联系方式" | 暗示导流 |

## ✅ 允许的引导动作(站内三件套)

| 动作 | 适合什么 | 例句 |
|---|---|---|
| **关注** | 系列 / 下一篇 | "想跟着看的,顺手关注一下" |
| **收藏** | 提示词图 / SOP / 干货 | "长按截图保存"或自然引导 |
| **评论** | 开放式问题 | "你最近用 AI 改稿改了几遍?评论区聊" |

## 边界判断

- ✅ 写"饶秋·1023"(小红书账号名,自我标识)
- ❌ 写"饶秋1023(公众号 / 视频号 / 小红书 / 抖音 全平台同名)"
- ⚠️ 公众号 / 视频号写作里饶秋习惯的"全网同名"口径**不能直接搬到小红书**

## 标签合规清单

**可用标签**(AI / 内容主题类):
- `#AI实战` `#提示词` `#AIGC`
- `#AI写作` `#DeepSeek` `#豆包`
- `#职场效率` `#知行合一`
- `#饶秋1023`(小红书账号名,可)

**不可用**:
- `#公众号` `#微信` `#视频号` `#抖音`
- `#加微信` `#引流` `#找我`

## CTA 模板(合规版)

参照 `templates/9-page-master.html` 的第 9 张 CTA:

```
· 一 句 收 尾 ·

判断力 > 提示词

(理由 38 px)

────

下一篇拆「让 AI 反问你」这一招,
想跟着看的,顺手关注一下。

你最近用 AI 写稿改了几遍?
哪一步最卡?评论区聊。
```

绝对不写:
- "今天就能带走的一句话"
- "不学就被淘汰"(贩卖焦虑)
- "喜欢就关注"(平庸)
- "转发给那个谁"(引流)
- 任何站外平台名

## 风险等级

| 等级 | 表现 | 后果 |
|---|---|---|
| 🟢 安全 | 只写 关注 / 收藏 / 评论 | 正常推流 |
| 🟡 警告 | 出现一次平台名(如"公众号同名") | 限流 1-3 天 |
| 🟠 严重 | "私信""加我"等诱导词 | 限流 7-30 天 |
| 🔴 封号 | 多平台名 + 引流套路 + 联系方式 | 封号 |
